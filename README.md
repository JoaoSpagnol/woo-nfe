
# WooCommerce NFe.io

WooCommerce NFe é uma extensão do WooCommerce para a API do NFe.

## Instalação

1. Vá no men *Plugins* e clique *Adicionar Novo*.
2. Pesquise por *WooCommerce NFe*.
3. Clique em *Instalar Agora*.
4. Ativar o plugin
